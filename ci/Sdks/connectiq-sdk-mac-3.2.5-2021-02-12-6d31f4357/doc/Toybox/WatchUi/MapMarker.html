Overview

The base class for the MapMarker object.

The MapMarker is used to hold a BitmapResource, or a MAP_MARKER_ICON_* enum value, and the corresponding x, y value to note the "hotspot" for the image to be used as an icon for a marker within a MapView. This class also contains the Location at which the MapMarker should be displayed on the map.

setIcon(icon, x, y)

Set a bitmap icon to use for the MapMarker object on a MapView.

Parameters:

  • icon — (Toybox.WatchUi.BitmapResource, Toybox.Lang.Number) —

    A BitmapResource or MAP_MARKER_ICON_* type

  • x — (Toybox.Lang.Number) —

    The x location of the hotspot on the BitmapResource in pixels (px). Used to align the hotspot with the longitude of the MapMarker. This value is required, but will be disregarded when used with MAP_MARKER_ICON_* type.

  • y — (Toybox.Lang.Number) —

    The y location of the hotspot on the BitmapResource in pixels (px). Used to align the hotspot with the latitude value of the MapMarker. This value is required, but will be disregarded when used with MAP_MARKER_ICON_* type.

Since:

3.0.0

Throws:

  • (Toybox.WatchUi.InvalidPointException) —

    Thrown if the x, y values for the hotspot fall outside the bounds of the icon image

  • (Toybox.Lang.UnexpectedTypeException) —

    Thrown if icon is not a supported type
