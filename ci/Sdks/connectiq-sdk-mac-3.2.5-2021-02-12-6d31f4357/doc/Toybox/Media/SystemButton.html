Overview

A SystemButton allows an app to override a default media player button. The button can be overridden by providing a new Toybox::WatchUi::BitmapResource or a Toybox::Lang::Number representing a 24-bit RRGGBB color value. If a color value is provided then the system default button will be used but will drawn using the given color.
